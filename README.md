## ‚ñ∂Ô∏è Getting Started

### Prerequisites
- Node.js, MongoDB, Git

### Installation

```bash
git clone https://github.com/yourusername/mern-chat-app.git
cd mern-chat-app

# Backend setup
cd backend
npm install

# Frontend setup
cd ../frontend
npm install
```
### Environment Variables
Create .env in backend 
```bash
PORT=5000
MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_secret
```
### Run the App
```bash
# Backend
cd backend
npm run dev

# Frontend
cd ../frontend
npm start
```
### Access the app at: http://localhost:3000
